# Instagram Cleaner

Instagramの画面上の特定のボタンやセクションを非表示にし、画面をシンプルにカスタマイズできるGoogle Chrome拡張機能です。

## 概要

この拡張機能は、Instagramのウェブ版において、自分が不要と感じるナビゲーション要素やセクションを個別にオン/オフして非表示にすることができます。Manifest V3に対応しており、軽量に動作します。

## Features

設定画面から、以下の要素の表示・非表示を切り替えることが可能です。

### セクション

- **フッター**: 画面下部のリンク集やメタ情報のフッターを非表示にします。

### ナビゲーションボタン

- **ホーム**
- **検索**
- **発見（Explore）**
- **リール**
- **メッセージ**
- **通知**
- **作成**
- **プロフィール**
- **バーガーメニュー（詳細）**
- **その他のMeta製アプリ**（Threadsなどへのリンク）

### フローティング要素
- **メッセージアイコン**: 画面右下に表示されるメッセージボタンを非表示にします。
 
## インストール方法（カスタマイズしたい方向け）

1. このリポジトリをクローンするか、ZIPファイルとしてダウンロードして解凍します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力して拡張機能管理画面を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、解凍したフォルダを選択します。

## How to Use

1. ChromeのツールバーにあるInstagram Cleanerをクリックして設定ポップアップを開きます。
2. 非表示にしたい項目のスイッチをオン（緑色）にします。
3. Instagramのページを開いている場合、設定は即座に反映されます。

## Tech Stack
- **Manifest V3**
- **JavaScript**: `content.js`, `popup.js`
- **CSS**: `style.css`
- **Storage API**
